# Babel Fish AI - امتداد تحويل الكلام إلى نص والترجمة باستخدام الذكاء الاصطناعي

<img src="images/icon128.png" alt="أيقونة Babel Fish AI" width="128" height="128">

**لاستخدام الامتداد، ستحتاج إلى مفتاح API من OpenAI (أو إلى مزود طرف ثالث إذا كنت تستخدم LiteLLM Proxy). يمكنك إنشاء مفتاح API من OpenAI هنا : [https://platform.openai.com/account/api-keys](https://platform.openai.com/account/api-keys)**

Babel Fish AI هو امتداد مبتكر لمتصفح Chrome صُمم في الأساس لتقديم تحويل صوتي قوي. حوّل صوتك إلى نص بدقة مذهلة بفضل واجهة برمجة التطبيقات Whisper من OpenAI، واستفد اختيارياً من الترجمة الآلية في الوقت الحقيقي. يمكنك استخدام Babel Fish AI حصرياً للتحويل أو تفعيل الترجمة الفورية حسب احتياجاتك.

[![CodeFactor](https://www.codefactor.io/repository/github/jls42/babelfishai/badge)](https://www.codefactor.io/repository/github/jls42/babelfishai)

## 🌟 الميزات

- **التحويل الصوتي المتقدم**
  - التقاط صوت عالي الجودة عبر ميكروفون جهازك.
  - تحويل دقيق يتم بواسطة واجهة برمجة التطبيقات Whisper من OpenAI.
  - دعم متعدد اللغات للتعرف على الصوت وعرض النص، مما يتيح تحويل المدخلات الصوتية بلغات مختلفة وعرض النتائج (التحويل والترجمة، إن تم تفعيلها) باللغة التي تختارها.
  - إدراج النص تلقائياً في الحقل النشط أو عرضه في مربع حوار مخصص.

- **الترجمة الآلية المدمجة (اختيارية)**
  - ترجمة فورية للنصوص المحولة إلى لغات متعددة، مع إمكانية التفعيل عند الحاجة.
  - استخدام نموذج ذكاء اصطناعي متقدم لضمان ترجمة دقيقة تعكس المعنى الأصلي.
  - حرية الاختيار لاستخدام التحويل فقط أو الجمع بين التحويل والترجمة.

- **واجهة مستخدم بديهية وقابلة للتخصيص**
  - وضع عرض مرن: منطقة إدخال نشطة أو نافذة حوار عائمة.
  - شريط حالة قابل للتخصيص مع اختيار الألوان والعتامة ومدة العرض.
  - اختصار لوحة مفاتيح (Ctrl+Shift+1 أو ⌘+Shift+1 على أجهزة Mac) لبدء/إيقاف التسجيل.
  - أيقونة مخصصة تتضمن ميكروفوناً والرقم "42" للتعرف الفوري.

- **خيارات متقدمة**
    - وضع الخبير لإعدادات مفصلة (عناوين URL الخاصة بواجهات API، التكوين حسب النطاق، إلخ).
    - إمكانية تخصيص نماذج التحويل والترجمة.
    - التوافق مع LiteLLM Proxy للاتصال بنماذج لغوية بديلة وتعطيل تسجيل الطلبات.
    - إدارة شاملة لعملية التدويل بفضل ملفات اللغة (_locales)، مما يوفر واجهة ودعم صوتي بعدة لغات.

## 🚀 الاستخدام مع LiteLLM Proxy

إذا كنت ترغب في استخدام نماذج لغوية غير OpenAI، يمكنك استخدام LiteLLM Proxy. إليك كيفية تهيئته:

Babel Fish AI متوافق مع [LiteLLM Proxy](https://litellm.ai/)، الذي يتيح استخدام نماذج لغوية بديلة مع واجهة API متوافقة مع تلك الخاصة بـ OpenAI.

### التهيئة

1.  **قم بتثبيت وتكوين LiteLLM Proxy:** اتبع التعليمات على موقع LiteLLM.
2.  **قم بتكوين امتداد Babel Fish AI:**
    *   في خيارات الامتداد (انقر بزر الماوس الأيمن على الأيقونة > خيارات)، فعّل "وضع الخبير".
    *   في قسم "التكوين المتقدم"، قم بتعديل **عنواوين URL الاثنين** الخاصة بواجهات API (Whisper والترجمة) للإشارة إلى مثيل LiteLLM Proxy الخاص بك (على سبيل المثال، `http://localhost:4000/v1/audio/transcriptions` و `http://localhost:4000/v1/chat/completions`). **استخدم نفس عنوان URL الأساسي للأثنين.**
    *   حدد الخيار "NoLog (LiteLLM فقط)" إذا كنت ترغب في تعطيل تسجيل الطلبات من قبل LiteLLM.

**مهم:** الخيار "NoLog" مصمم **حصرياً** للاستخدام مع LiteLLM Proxy. **لا تقم بتفعيله** إذا كنت تستخدم واجهة API الرسمية لـ OpenAI، حيث سيؤدي ذلك إلى حدوث خطأ ويمنع الترجمة من العمل.

## 🛠️ الآلية التقنية

### بنية الامتداد

يتكون الامتداد من عدة ملفات JavaScript تتفاعل مع بعضها البعض:

*   **`manifest.json`:** ملف التكوين الرئيسي للامتداد. يحدد الأذونات، السكريبتات، الموارد المتاحة، إلخ. يستخدم الإصدار 3 من البيان ويعلن عن الأذونات `activeTab`، `storage`، `commands` و `scripting`. يتم حقن سكريبتات المحتوى (`content_scripts`) في جميع عناوين URL وتُنفذ في نهاية تحميل المستند.
*   **`background.js`:** عامل الخدمة الذي يعمل في الخلفية. يدير الأحداث (النقر على الأيقونة، اختصارات لوحة المفاتيح)، يُحقن سكريبت المحتوى إذا لزم الأمر، ويتواصل مع سكريبت المحتوى.
*   **`content.js`:** السكريبت الذي يتم حقنه في صفحات الويب. يتفاعل مباشرة مع DOM، يلتقط صوت الميكروفون، يستدعي واجهات API للتحويل والترجمة، ويعرض النتائج.
*   **`src/utils/api.js`:** يحتوي على الدالة `transcribeAudio` لاستدعاء واجهة API الخاصة بـ Whisper من OpenAI (التحويل).
*   **`src/utils/translation.js`:** يحتوي على الدالة `translateText` لاستدعاء واجهة API الخاصة بـ GPT من OpenAI (الترجمة).
*   **`src/utils/ui.js`:** يحتوي على دوال مساعدة لإدارة واجهة المستخدم (الشريط، مربع الحوار، زر النسخ).
*   **`src/constants.js`:** يحدد الثوابت الخاصة بالتكوين، الحالات، الإجراءات، إلخ.
*   **`src/pages/options/`:** يحتوي على الملفات الخاصة بصفحة خيارات الامتداد (HTML، CSS، JavaScript).

### عملية التحويل والترجمة

1.  **بدء التسجيل:** يبدأ المستخدم التسجيل بالنقر على أيقونة الامتداد أو باستخدام اختصار لوحة المفاتيح. يرسل سكريبت الخلفية رسالة إلى سكريبت المحتوى لبدء التسجيل.
2.  **التقاط الصوت:** يستخدم سكريبت المحتوى واجهة API `navigator.mediaDevices.getUserMedia` للوصول إلى الميكروفون وتسجيل الصوت.
3.  **التحويل:** يستخدم سكريبت المحتوى الدالة `transcribeAudio` (`src/utils/api.js`) لإرسال الصوت إلى واجهة API الخاصة بـ Whisper من OpenAI. تقوم الواجهة بإرجاع النص المحول.
4.  **الترجمة (اختيارية):** إذا كانت ميزة الترجمة مفعلة، يستخدم سكريبت المحتوى الدالة `translateText` (`src/utils/translation.js`) لإرسال النص المحول إلى واجهة API الخاصة بـ GPT من OpenAI. تقوم الواجهة بإرجاع النص المترجم.
5.  **العرض:** يعرض سكريبت المحتوى النص المحول (والنص المترجم إن وُجد) إما في العنصر النشط في الصفحة (إذا كان حقل نصي أو عنصراً قابلاً للتعديل) أو في مربع حوار.

### التواصل

يتم التواصل بين سكريبت الخلفية وسكريبت المحتوى عبر واجهة الرسائل في Chrome (`chrome.runtime.sendMessage` و `chrome.runtime.onMessage`).

### تخزين البيانات

يستخدم الامتداد `chrome.storage.sync` لتخزين:

*   مفتاح API الخاص بـ OpenAI (`apiKey`).
*   خيارات الامتداد (العرض، الترجمة، ألوان الشريط، إلخ).

يتم تخزين هذه البيانات محلياً على جهاز الكمبيوتر الخاص بك، ضمن تخزين امتداد Chrome.

### إدارة الأخطاء

الأخطاء المحتملة (مفتاح API مفقود، خطأ في التحويل، إلخ) مُعرّفة في ملف `constants.js`. تدير الدوال في `api.js` و `translation.js` الأخطاء المحتملة لاستدعاءات API. يعرض سكريبت المحتوى رسائل الأخطاء للمستخدم عبر شريط يظهر أعلى الصفحة.

## 🛡️ الأمان والخصوصية

- **حماية البيانات:**
  - يتم تخزين مفتاح API بطريقة آمنة داخل Chrome.
  - لا يحتفظ الامتداد ببياناتك الصوتية؛ جميع العمليات تتم في الوقت الحقيقي.
  - يتم التواصل مع واجهات API عبر اتصالات HTTPS الآمنة.

لمزيد من المعلومات حول كيفية إدارة بياناتك بواسطة BabelFishAI، يرجى مراجعة [سياسة الخصوصية](PRIVACY.md).
